# File Browser Component

一个用于浏览和管理服务器文件系统的工作台组件。

## 功能

- 📁 浏览服务器文件目录
- 📄 查看文件内容
- 🔐 密码认证保护
- 📋 复制文件路径
- 🔄 刷新目录列表
- ⬆️ 返回上级目录
- 🚪 安全登出

## 布局设计

由于工作台组件只显示在右侧半屏，采用了以下设计：

1. **紧凑的头部** - 包含面包屑导航和操作按钮
2. **单面板切换** - 通过按钮在文件列表和文件内容之间切换
3. **优化间距** - 所有元素使用紧凑的内边距
4. **图标按钮** - 使用图标代替文字按钮，节省空间
